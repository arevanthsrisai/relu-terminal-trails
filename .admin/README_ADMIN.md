# Terminal Trials administration

This folder is for the game author only.

- Access keys are stored only in each sector's hidden `.metadata.conf`.
- `SOLUTIONS.txt` is an intentionally empty-content placeholder for author notes.

The current event build contains seven playable sectors. Authors maintain
`.master/LevelNN/.metadata.conf` with the sector access key. `.master/` is the
template area; players work only under `runtime/current`.
